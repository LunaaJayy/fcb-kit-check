

## 🧞 Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Installs dependencies                            |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
| `npm run build`           | Build your production site to `./dist/`          |
| `npm run preview`         | Preview your build locally, before deploying     |
| `npm run astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `npm run astro -- --help` | Get help using the Astro CLI                     |

## 👀 Want to learn more?

Feel free to check [our documentation](https://docs.astro.build) or jump into our [Discord server](https://astro.build/chat).

**Wat heb ik vandaag gedaan?** \
Vandaag hebben we de eerste helft van de dag geluisterd naar de introductie en een kleine workshop over Astro. Daarna heb ik bedacht wat ik wil gaan doen: welke content api's ik wil gebruiken, welke web api ik ga gebruiken, en wat mijn concept precies is:
Een webpagina waarbij je de aankomende wedstrijden van FC Barcelona kan zien, het weer wt het dan wordt en welke 'kit' (team tenue, merchandise van het team) je dan het beste aan kan doen. Voor de kit heb ik geen API, die foto's moet ik zelf vinden. De detailpagina zal een overzicht worden van de statistieken tussen de twee teams.

De webapi die ik zal gebruiken is de geolocation API, voor als de gebruiker de wedstrijd thuis wilt kijken.
